Question 1)

Palindrome Checker

Given a string word, the task is to find whether the string is a palindrome or not without using library methods.

Start out by determining what the question is asking. A palindrome is defined as a word that spells the same forwards and backwards. In order to determine if the given String is a palindrome or not, we will need to view each character individually by using a for loop. Characters in a String start at index 0 and end at index lengthOfString - 1. Since palindromes can be spelled the same forward and backward, an example being "racecar", we can stop our for loop halfway through the String's length (line 10). Inside the for loop, we want to compare the first letter with the last letter, and move inwards from there. To do that, we can use an if statement that determines whether or not those two characters are unequal (line 11). If they are unequal, we can stop our program and return false and conclude that the input String is not a palindrome.
